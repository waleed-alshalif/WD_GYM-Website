A modern, high-performance, and fully responsive fitness website designed for professional gyms and training centers. **WD_GYM** provides a sleek user interface to showcase trainers, services, and fitness plans.

---

## ✨ Key Features

### 📱 Fully Responsive Design
- Optimized for desktops, tablets, and mobile devices using **Bootstrap**.
- Fluid layouts and flexible components ensure a consistent experience across all screen sizes.

### 👥 Trainer Showcases
- Dedicated section to highlight expert gym trainers with social media integration.
- Professional profiles with clear calls to action.

### 🏆 "Why Choose Us" Section
- Interactive cards showcasing the unique selling points of the gym, including quality equipment, nutrition plans, and healthy diet tracking.

### 📩 Integrated Contact System
- Built-in contact form for direct user engagement.
- Location and contact information sections for easy accessibility.

### 🎨 Modern Aesthetics
- Clean typography and high-quality imagery.
- Smooth carousel transitions for a dynamic hero section.

## 🚀 Getting Started

### 📋 Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari).
- (Optional) A local server environment like **XAMPP** or **WAMP** if you wish to use the PHP contact functionality.

### ⚙️ Installation
1. **Clone the Repository**
   ```bash
   git clone https://github.com/waleed-alshalif/WD_GYM-Website.git
   ```
2. **Open the Project**
   - Navigate to the project folder.
   - Open `index.html` in your preferred browser.
